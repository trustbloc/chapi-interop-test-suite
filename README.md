# chapi-interop-test-suite

This repository contains the Draft W3C
[Credential Handler API](https://w3c-ccg.github.io/credential-handler-api/) interoperability test suite.

There are multiple scenarios, each of which is detailed below.

## Scenarios

## Running Locally
```sh
npm install
npm test
```

Note: current version of tests are running against [trustbloc local](https://github.com/trustbloc/sandbox) version only.
